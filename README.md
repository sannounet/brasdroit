# Bras Droit — Logiciel de gestion integree

## Structure
- `/backend` → FastAPI + PostgreSQL → deployer sur Render
- `/frontend` → HTML/CSS/JS → deployer sur Vercel

## Deploiement rapide
1. Pousser ce repo sur GitHub
2. Render : connecter `/backend` → render.yaml
3. Vercel : connecter `/frontend`
4. Mettre à jour API_URL dans frontend/index.html
